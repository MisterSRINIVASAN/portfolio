## 🌟 About The Project

This is my personal developer portfolio, meticulously designed to showcase my journey as a **Full-Stack Developer**, **AI/ML Engineer**, **ML Enthusiast**, and **Data Analyst**. 

The application utilizes a dark, premium aesthetic infused with vibrant, dynamic neon accents (Cyan, Fuchsia, Emerald). It features advanced CSS techniques like **Glassmorphism**, dynamic 3D perspective tilt on hover, and smooth scrolling animations powered by Framer Motion.

### ✨ Key Features

- **Dynamic Theming:** Project cards and technical skill categories automatically cycle through a rich palette of neon accent colors.
- **3D Interactions:** Cards physically tilt and react to mouse movements to create a deep, immersive spatial effect.
- **Glassmorphism Design:** Frosted glass elements overlaying a custom grid and ambient animated background glows.
- **Fully Responsive:** Beautifully scales from large desktop monitors down to mobile devices.
- **Dynamic Typing:** Hero section features an automatic typewriter effect.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have Node.js and npm installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MisterSRINIVASAN/portfolio.git
   ```
2. Navigate to the project directory
   ```sh
   cd portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm start
   ```
